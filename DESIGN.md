# Design

## Visual Theme
Dark, aggressive, premium "bro marketing" feel. High contrast, clean boxes with subtle glows, and handwritten accents.

## Colors
- Main background: `#050505` (near-black)
- Card background: `#0B0B0D`
- Elevated card background: `#101014`
- Primary red: `#FF2538`
- Dark red: `#6D111B`
- White: `#F5F5F5`
- Muted text: `#8D8D94`
- Border: `rgba(255,255,255,0.08)`

## Typography
- **Heading Display**: Oswald (condensed, bold, uppercase)
- **Accent Handwritten**: Permanent Marker (bold handwritten brush-style for focus words)
- **Body Text**: Inter (clean sans-serif)

## Layout & Components
- **Global Header**: Sticky transparent header with glassmorphism (backdrop-blur) on scroll.
- **Hero Mockup**: Pure HTML/CSS floating smartphone mockup showcasing the dashboard.
- **Service Cards**: Hover interactions, subtle red glow, elevated borders.
- **Process Accordion/Tabs**: Vertical step layout with animated content transitions.
- **Final CTA**: Centered action block with dark layout and strong red radial glow.
